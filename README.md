# IRONCORE — Landing Page (Campaign)

Modern high-performance e-commerce landing page for the IRONCORE athletic wear brand.  
Designed with a focus on conversion, bold branding, and a premium fitness aesthetic.

---

## 🚀 Overview

IRONCORE is a fictional athletic performance brand landing page built to simulate a real-world product drop campaign.  
The project focuses on strong visual hierarchy, conversion-driven design, and a modern fitness/streetwear brand identity.

---

## 🎯 Key Features

- Fully responsive landing page design
- Hero section with video background
- Countdown timer for limited drop campaign
- Product showcase grid
- Discount and urgency-driven marketing sections
- Customer testimonials
- Multi-language support structure (ET / EN ready)
- Modular i18n system using `data-i18n` attributes

---

## 🧱 Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Simple i18n translation system
- Responsive layout (mobile-first approach)

## 🛍️ Project Concept

This project simulates a limited-time product drop campaign for a fictional sportswear brand.  
The goal was to replicate modern e-commerce landing pages used by brands like Gymshark, Nike, and Under Armour.

Focus areas:
- urgency-driven UX (limited drop, countdown)
- conversion-focused copywriting
- premium minimalist branding style
- fast and clean UI structure
